# 공사 공정표 웹앱

GitHub Pages 배포용 정적 HTML 웹앱입니다.

## 배포 방법

1. 이 폴더 안의 `index.html`과 `.nojekyll` 파일을 GitHub 저장소 루트에 올립니다.
2. GitHub 저장소에서 Settings → Pages로 이동합니다.
3. Build and deployment에서 Source를 `Deploy from a branch`로 선택합니다.
4. Branch를 `main`, Folder를 `/ (root)`로 선택하고 Save합니다.
5. 배포가 끝나면 `https://계정명.github.io/저장소명/` 주소로 접속합니다.
